## Linux Basic Commands:

**switch to root user**
````
sudo -i
````
**shows username**
````
whoami
````
**shows hostname**
````
hostname
````
**shows present working dir**
````
pwd
````
**display os informaton**
````
cat /etc/os-release
````
**display kernel information**
````
uname -a
````
**display memory information**
````
free -h
````
**display disk information**
````
df -h
````

**list files and dir's**
````
ls
````

**change directory**
````
cd <dirname>
````
**back to previous dir**
````
cd ..
````
